# TFG DAM - Plataforma de Alquiler de Objetos entre Particulares

## Título del TFG

Plataforma de alquiler de objetos entre particulares con arquitectura cliente-servidor en Java

## Breve descripción del proyecto

Este proyecto consiste en el desarrollo de una plataforma digital que permite a usuarios particulares publicar objetos y alquilarlos a otras personas de manera segura y controlada.
El sistema se implementará con un backend en Spring Boot (Java), un cliente de escritorio en JavaFX y una base de datos relacional.
