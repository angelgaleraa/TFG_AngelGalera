# TFG DAM - Plataforma de Alquiler de Objetos entre Particulares

## Título del TFG

Plataforma de alquiler de objetos entre particulares con arquitectura cliente-servidor en Java

## Breve descripción del proyecto

## 1. Resumen

Este proyecto consiste en el desarrollo de una plataforma digital orientada al alquiler de objetos entre particulares, cuyo objetivo principal es facilitar el acceso temporal a bienes de uso ocasional de forma segura, organizada y eficiente, reduciendo así la necesidad de compra de productos que solo se utilizan de manera puntual.

El sistema se basa en una arquitectura cliente-servidor, en la que la capa de presentación se implementa mediante una aplicación de escritorio desarrollada en JavaFX, mientras que la lógica de negocio y el acceso a datos se gestionan a través de un backend desarrollado con Spring Boot en Java. La información se almacena en una base de datos relacional, permitiendo la persistencia y gestión estructurada de usuarios, objetos, reservas y valoraciones. La comunicación entre el cliente y el servidor se realiza mediante servicios REST en formato JSON, aplicando una separación clara de responsabilidades y buenas prácticas de diseño de software.

El alcance del proyecto abarca la implementación de un sistema completo de gestión de alquileres entre particulares. Entre las funcionalidades clave se incluyen el registro e inicio de sesión de usuarios, la publicación y edición de objetos disponibles para alquilar, la búsqueda y filtrado por diferentes criterios, la gestión de reservas con control de fechas y estados, un sistema de mensajería interna para la comunicación entre usuarios y un módulo de valoraciones que permite evaluar la experiencia tras la finalización de cada alquiler. Asimismo, se incorpora la simulación de transacciones económicas con el fin de registrar los pagos y generar comprobantes asociados a las reservas realizadas.

Desde una perspectiva técnica y formativa, el proyecto permite aplicar conocimientos relacionados con la programación orientada a objetos, el desarrollo de aplicaciones distribuidas, la integración de bases de datos mediante JPA/Hibernate y el uso de herramientas de control de versiones. De este modo, la plataforma no solo responde a una necesidad real en el ámbito de la economía colaborativa, sino que también constituye un entorno práctico para la consolidación de competencias propias del desarrollo de aplicaciones multiplataforma en un contexto profesional.
